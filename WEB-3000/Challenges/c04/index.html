The :not Selector

Because giving others what you don't want is fun...

The :not CSS selector will apply to everything else NOT what you have specified.

For example

:not(blockquote) {
font-family: Avenir, sans-serif;
color: #000;
}

So as you can see, everything else besides the blockquote is Avenir (blockquote is set to Helvetica) or sans-serif; based off the browser.

Now lets get fancy, and not just regular fancy, but Canadian fancy. You can also designate a selector, for our purposes an li with a class, to be singled out of the styling from the rest.

Exhibit A...eh

li:not(.somethingelse) {
color: red;
)

  • I don't want this color, I'm going to dump it on all these other chumps
  • I'll take some also jerk face li tag

Here we are making every other li font color red except the li with the class somehtingelse. If that doesn't scream maple leaf magic I don't know what does.
